# Textify - 제품 기획서

> 이미지를 텍스트 아트로 변환하는 iOS 앱

---

## 1. 제품 개요

### 1.1 비전
**"모든 이미지를 텍스트 아트로"** - 사진을 ASCII/유니코드 텍스트 아트로 변환하여 메신저, SNS, 코드 주석 등에서 창의적으로 활용

### 1.2 타겟 사용자
| 페르소나 | 사용 목적 |
|----------|----------|
| 개발자 | 코드 주석, README 장식 |
| SNS 크리에이터 | 독특한 텍스트 기반 콘텐츠 |
| 메신저 사용자 | 텍스트로 공유 가능한 이미지 |
| 레트로 애호가 | ASCII art 감성 |

### 1.3 핵심 가치
- **즉시성**: 사진 선택 → 3초 내 결과
- **공유 용이성**: 텍스트로 어디서든 복사/붙여넣기
- **커스터마이징**: 문자, 폭, 대비 등 세부 조정

## 2. 기능 명세

### 2.1 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 이미지 선택 | 사진 라이브러리/파일에서 이미지 가져오기 | P0 |
| 텍스트 변환 | 이미지 → ASCII/유니코드 아트 변환 | P0 |
| 문자 팔레트 | 사용할 문자 세트 직접 입력 | P0 |
| 결과 복사 | 클립보드에 텍스트 복사 | P0 |
| 결과 저장 | 이미지로 내보내기 | P1 |
| 히스토리 | 이전 변환 결과 저장/조회 | P1 |
| 설정 | 다크모드, 기본값 설정 | P2 |

### 2.2 변환 옵션

| 옵션 | 범위 | 기본값 | 설명 |
|------|------|--------|------|
| 출력 폭 | 20-200자 | 80 | 한 줄당 문자 수 |
| 밝기 반전 | On/Off | Off | 어두운 배경용 |
| 대비 부스트 | 0.5-2.0 | 1.0 | 밝기 차이 강조 |

### 2.3 문자 팔레트 프리셋

```
기본:     @%#*+=-:.
확장:     $@B%8&WM#*oahkbdpqwmZO0QLCJUYXzcvunxrjft/\|()1{}[]?-_+~<>i!lI;:,"^`'.
블록:     █▓▒░
미니멀:   #.
숫자:     0123456789
커스텀:   (사용자 입력)
```
